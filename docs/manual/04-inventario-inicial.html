¿Para qué sirve?

El módulo de Inventario inicial te permite cargar el stock existente cuando arrancás a usar el sistema por primera vez. Si ya tenés productos en tu depósito antes de registrar la primera compra, aquí es donde declarás cuántas unidades hay y a qué costo las adquiriste.

Este módulo no reemplaza las compras ni los ajustes de stock del día a día. Solo se usa una vez, al inicio. Después de eso, el stock se mueve con las compras, ventas y ajustes normales del sistema.

Pasos

  1. 1

    Abrir el módulo

    En el menú lateral, andá a Inventario → Inventario inicial. Esta opción solo es visible para usuarios con rol administrador.

    Al cargar la pantalla vas a ver el nombre del depósito activo —por ejemplo, Depósito: Depósito principal— y la tabla con todos los productos que tienen seguimiento de stock activado.

  2. 2

    Entender la tabla

    La tabla tiene una fila por producto con las siguientes columnas:

    SKU Producto Unidad Cantidad inicial Costo unitario
    CART-001 Caja de cartón UN campo editable campo editable
    FILM-002 Film stretch RLL 🔒 Con stock —

    Las filas marcadas con 🔒 Con stock ya tienen movimientos registrados y no pueden modificarse desde aquí.

  3. 3

    Completar cantidad y costo

    Para cada producto que ya tenés en depósito, ingresá en la columna Cantidad inicial cuántas unidades hay, y en Costo unitario cuánto pagaste por cada una.

    Los productos que todavía no tenés en stock podés dejarlos en blanco: si la cantidad queda vacía o en 0, ese producto se omite.

    ⚠ El costo unitario es obligatorio si completás la cantidad Si ingresás una cantidad pero dejás el costo en blanco, el sistema va a mostrar el error "Requerido" al intentar guardar. Si el costo real era cero (donación, muestra gratis), ingresá 0 explícitamente.
  4. 4

    Revisar antes de guardar

    Antes de confirmar, repasá que las cantidades y costos corresponden al stock real del día. Una vez guardado, esos datos no se pueden editar desde esta pantalla.

    ⚠ El inventario inicial solo se aplica una vez por producto En cuanto guardás, el producto queda bloqueado en esta pantalla (aparece con el candado 🔒 Con stock). Si después necesitás corregir una cantidad, tenés que usar el módulo de Ajustes de stock — no el inventario inicial.
  5. 5

    Guardar el inventario

    Hacé click en el botón Cargar inventario inicial en la parte inferior derecha de la pantalla.

    Si todo está correcto, aparece el mensaje "Inventario inicial cargado correctamente." en verde. Los productos cargados pasan automáticamente al estado 🔒 Con stock y ya no se pueden tocar desde aquí.

Consejos / Errores comunes

  • Cargá el inventario inicial antes de la primera compra o venta Una vez que registrás una compra o una venta para un producto, ese producto queda bloqueado en esta pantalla. Si querés que el stock arranque desde un valor distinto de cero, hacelo acá antes de operar con él.
  • Solo aparecen productos con seguimiento de stock activado Si un producto no aparece en la tabla, revisá que tenga activado el toggle "Controlar stock de este producto" en su ficha de producto. Los productos sin seguimiento de stock no generan movimientos y no se gestionan desde aquí.
  • Error: "ya tiene movimientos de stock registrados" Si al guardar aparece el mensaje "[nombre del producto] ya tiene movimientos de stock registrados. Quitá la cantidad para ese producto.", significa que ese producto ya fue usado en una compra o venta. Borrá la cantidad ingresada para esa fila y volvé a intentar. Para ajustar su stock usá Ajustes de stock.
  • Cometiste un error en el costo o la cantidad ya guardada El inventario inicial no se puede deshacer desde esta pantalla. Para corregir una cantidad equivocada, usá el módulo de Ajustes de stock (sumar o restar la diferencia). Para corregir un costo, comunicate con tu administrador: puede requerir un ajuste manual en el sistema.
  • No se encontró un depósito activo Si ves el mensaje "No se encontró un depósito activo. Verificá la configuración de depósitos.", significa que no hay ningún depósito habilitado en el sistema. Contactá a tu administrador para que configure un depósito antes de continuar.
  • Podés cargar solo algunos productos ahora y el resto después No es necesario completar todos los productos de una sola vez. Podés cargar los productos que tenés hoy, y cuando recibas más stock nuevo usás el módulo de Compras para registrar esa entrada. El inventario inicial es solo para el stock que ya tenías antes de arrancar con el sistema.

Ante cualquier duda, contactá a tu administrador. · Volver al índice
