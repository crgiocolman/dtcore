Manuales de usuario

Seleccioná el módulo sobre el que necesitás ayuda.

Guía Rápida

Primera vez que usás el sistema
1
Crear categorías y productos
Ir a Productos → Nuevo. Completar nombre, unidad de medida y precio de venta.
2
Cargar stock inicial
Si ya tenés mercadería en tu negocio, cargá la cantidad y costo actual de cada producto. Esto se hace una sola vez.
3
Registrar una venta
Ir a Ventas (o POS). Buscar el producto, indicar cantidad y confirmar.
Operación del día a día
1
Registrar una compra
Cuando recibís mercadería de un proveedor, registrala en Compras → Nueva compra. El stock se actualiza solo al confirmar.
2
Registrar una venta
Ir a Ventas (o POS). Buscar el producto, indicar cantidad y confirmar.
3
Ver reportes
Revisá ventas del día, stock actual y ganancia estimada en Reportes.

Conceptos clave

Costo Promedio (CPP)
El sistema calcula cuánto cuesta en promedio cada unidad considerando todas las compras.
Ajuste de stock
Para cargar inventario inicial o corregir diferencias con el stock físico.
Cancelación
Anula una compra o venta. El stock y CPP se restauran al estado anterior.

Consejos clave

  • ✅ Registrá cada compra — el stock siempre estará actualizado.
  • ✅ El CPP lo calcula el sistema solo.
  • ✅ Antes de vender, verificá que haya stock suficiente.
  • ✅ Podés cancelar una compra o venta desde el detalle.
  • ⚠️ El stock en negativo está desactivado — el sistema avisa si no hay suficiente.
